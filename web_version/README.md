# PAULISTA PDV - Sistema de Vendas

## Instalação Rápida

### Passo 1: Instalar Node.js (apenas uma vez)
1. Baixe de: https://nodejs.org
2. Execute o instalador
3. Clique em "Next" até finalizar

### Passo 2: Instalar o Sistema
1. Execute `INSTALAR.bat`
2. Aguarde a instalação terminar
3. Atalhos serão criados no Desktop

### Passo 3: Usar o Sistema
- **PC Principal (Servidor):** Execute `Paulista PDV - Servidor`
- **Outros PCs (Clientes):** Execute `Paulista PDV - Cliente`

---

## Arquivos Importantes

| Arquivo | Função |
|---------|--------|
| `INSTALAR.bat` | Instala o sistema |
| `INICIAR_PAULISTA_PDV.bat` | Inicia o servidor |
| `ACESSAR_REMOTO.bat` | Conecta de outro PC |
| `.env.local` | Configurações do banco de dados |

---

## Suporte

Todos os dados são salvos na nuvem automaticamente.
Qualquer PC com esta pasta terá acesso aos mesmos dados.
